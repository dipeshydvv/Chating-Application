# 📚 Study Mode - Quick Reference Card

## 🚀 Quick Start

### Add Participants in 3 Steps

```
1. Click 📚 Study Mode icon
   ↓
2. Start study session (topic + duration)
   ↓
3. Type name → Click Add (or press Enter)
```

---

## 📋 Study Mode Features

| Feature | How to Use |
|---------|-----------|
| **Start Session** | Enter topic → Set duration → Click Start |
| **Add Participant** | Type name → Click Add or press Enter |
| **Remove Participant** | Hover over tag → Click × |
| **Add Goal** | Type goal → Click Add or press Enter |
| **Complete Goal** | Click checkbox next to goal |
| **Enable Focus Mode** | Check the Focus Mode checkbox |
| **End Session** | Click "End Study Session" button |

## 💡 Tips

✅ **Use Clear Names**
- "Rahul" (good)
- "R" (bad)

✅ **Set Specific Goals**
- "Complete Chapter 5 exercises" (good)
- "Study hard" (bad)

✅ **Enable Focus Mode**
- Reduces distractions
- Better concentration
- Improves productivity

✅ **Add Relevant Participants**
- Only active members
- Keep group focused
- Remove inactive members

---

## ❌ Common Mistakes

| Mistake | Solution |
|---------|----------|
| Can't add participant | Start session first |
| Duplicate participant | System prevents it |
| Participant disappeared | Session ended (cleared) |
| Can't remove participant | Hover first, then click × |

---

## 🆘 Troubleshooting

**Problem:** Can't add participant
- **Solution:** Make sure study session is active

**Problem:** Participant name not showing
- **Solution:** Check if already added (no duplicates)

**Problem:** Participants disappeared
- **Solution:** They're cleared when session ends

**Problem:** Can't remove participant
- **Solution:** Hover over tag first, then click ×
