## 🔧 Technical Details

### Component: SplashScreen.js
```javascript
- State management for typing animation
- useEffect for character animation
- useEffect for message rotation
- CSS animations via style tag
- Responsive Tailwind classes
```

### Key Functions:
```javascript
// Typing animation effect
- Displays one character at a time
- 50ms delay between characters
- 2s pause between messages

// Message rotation
- Cycles through 6 messages
- Auto-advances after typing
- Infinite loop
```

### CSS Animations:
```css
@keyframes blob - Floating blob motion
@keyframes float - Chat bubble floating
@keyframes gradient - Color gradient shift
```
