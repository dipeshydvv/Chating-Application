# Quick Guide - Image & Video Sharing

## ✅ Feature Added

Image and video sending in personal one-to-one DMs for every user.

---

## 🎯 How to Use

### Send an Image:
1. Open chat with contact
2. Click 🖼️ Image button (pink hover)
3. Select image file
4. Image appears in chat ✅

### Send a Video:
1. Open chat with contact
2. Click 🎥 Video button (cyan hover)
3. Select video file
4. Video appears with player ✅

---

## 📊 What's Supported

### Images:
- JPG, PNG, GIF, WebP, BMP, SVG, TIFF, ICO

### Videos:
- MP4, WebM, MOV, AVI, MKV, FLV, WMV, 3GP

---
