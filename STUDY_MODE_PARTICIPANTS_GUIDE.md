# 📚 Study Mode - Adding Learning Participants Guide

## How to Add Learning Participants in Study Mode

### Step 1: Start a Study Session
1. Click the 📚 **Study Mode** icon in the chat header
2. Enter a **Study Topic** (e.g., "Mathematics Chapter 5")
3. Set the **Planned Duration** (in minutes)
4. Click **"Start Study Session"** button

### Step 2: Add Participants
Once the study session is active, you'll see the **Study Group** section at the bottom:

1. **Enter Participant Name**
   - Type the name of the participant in the input field
   - Example: "Rahul", "Priya", "Alex", etc.

2. **Add the Participant**
   - Click the **"Add"** button, OR
   - Press **Enter** key on your keyboard

3. **Participant Added Successfully**
   - The participant will appear as a blue tag with their name
   - Participant count will increase

### Step 3: Manage Participants

#### Remove a Participant
1. Hover over a participant tag
2. Click the **×** button that appears
3. Participant will be removed from the study group

#### Add More Participants
- Repeat the process as many times as needed
- Add as many participants as you want
- Each participant will be tracked in the study session

### Features

✅ **Add Multiple Participants**
- Add as many learning participants as needed
- No limit on the number of participants

✅ **Remove Participants**
- Hover over any participant tag to see the remove button
- Click × to remove them from the study group

✅ **Duplicate Prevention**
- System prevents adding the same participant twice
- You'll get an alert if you try to add a duplicate

✅ **Real-time Tracking**
- Participant count updates instantly
- All participants are saved in the study session

✅ **Persistent Storage**
- Participants are saved locally
- When you reopen the study session, all participants are still there

## FAQ

### Q: Can I add participants before starting the session?
**A:** No, you can only add participants after starting the study session. The first participant is automatically added when you start.

### Q: Can I add the same person twice?
**A:** No, the system prevents duplicate participants. You'll get an alert if you try.

### Q: Are participants saved after I end the session?
**A:** No, participants are only saved during the active session. When you end the session, the participant list is cleared for the next session.

### Q: Can I add participants from my contacts?
**A:** Currently, you need to manually type participant names. In the future, we'll add contact integration.

### Q: How many participants can I add?
**A:** There's no limit! Add as many as you need.

### Q: Can participants see each other?
**A:** Yes, all participants in the study group can see each other's names in the Study Group section.

### Q: Can I edit a participant's name?
**A:** Currently, you need to remove and re-add with the correct name. We're working on edit functionality.

### Q: What happens to participants when I close the modal?
**A:** Participants are saved in your study session. When you reopen Study Mode, they'll still be there.

## Troubleshooting

### Issue: Can't add participant
**Solution:** Make sure you've started the study session first.

### Issue: Participant name doesn't appear
**Solution:** Check if the name was already added (no duplicates allowed).

### Issue: Participants disappeared
**Solution:** They're only saved during the session. If you ended the session, they were cleared.

### Issue: Can't remove participant
**Solution:** Hover over the participant tag first, then click the × button that appears.
