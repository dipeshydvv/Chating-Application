### CSS Example:
```css

  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 20px;
}

/* Responsive design */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

/* Animations */
@keyframes slideIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.element {
  animation: slideIn 0.5s ease-out;
}
```

## 💡 Tips for Learning

### HTML Tips:
```
1. Use semantic tags (header, nav, main, footer)
2. Always include meta viewport tag
3. Use alt text for images
4. Validate your HTML
5. Follow accessibility guidelines
```

### CSS Tips:
```
1. Use classes for styling, not IDs
2. Mobile-first approach
3. Use CSS variables for consistency
4. Avoid inline styles
5. Test on multiple browsers
```

### Java Tips:
```
1. Follow naming conventions
2. Use meaningful variable names
3. Write small, focused methods
4. Handle exceptions properly
5. Use design patterns
```
