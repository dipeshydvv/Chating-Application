# 🔐 Real User Verification - Quick Start

## 30-Second Setup

### 1. Register a User
```
Click: 👥 User Registration (top-right header)
Click: "Register New User"
Fill: Username, Email, Password
Click: "Register User"
```

### 2. Add to Study Session
```
Click: 📚 Study Mode
Enter: Topic & Duration
Click: "Start Study Session"
Type: Registered username
Click: "Add"
✅ User added with verification!
```
