# 📺 Media Sharing Quick Guide

## How to Share Audio & Video

### Step 1: Open Co-Watching/Co-Listening
1. Select a contact from the sidebar
2. Click the 👁️ **Co-Watching** icon in the chat header
3. A modal will open with the media player

### Step 2: Add Media
1. Scroll to **"Add to Playlist"** section
2. Select media type:
   - 🎬 **Video** - YouTube, MP4, WebM, MOV
   - 🎵 **Audio** - MP3, WAV, OGG
   - 📄 **Document** - PDF, DOC, DOCX

3. Enter a **Title** (e.g., "Python Tutorial")
4. Paste the **URL** (see examples below)
5. Click **"Add to Playlist"** button

### Step 3: Play Media
1. Click on any media item in the playlist
2. Use the **Play/Pause** button to control playback
3. The media will appear in the player area

### Step 4: Share with Contact
1. Click the **Share** button (blue button)
2. You'll see a confirmation: "📤 'Title' shared with Contact!"
3. The media will appear in the chat with a full player

### Step 5: View in Chat
- The shared media appears in the chat with:
  - Media title
  - Full audio/video player
  - Timestamp
  - All message features (reactions, forward, etc.)

## 📋 Supported URLs

### YouTube Videos
```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
https://youtu.be/dQw4w9WgXcQ
```
✅ Automatically converted to embed format

### Audio Files
```
https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3
https://example.com/music/song.mp3
https://example.com/podcast.wav
https://example.com/audio.ogg
```

### Video Files
```
https://example.com/video.mp4
https://example.com/movie.webm
https://example.com/clip.mov
https://example.com/video.avi
```

### Direct Links
- Any direct URL to audio/video file
- Must be publicly accessible
- CORS enabled for best compatibility

## ❓ FAQ

### Q: What if the URL doesn't work?
**A:** Make sure:
- URL is publicly accessible
- URL is correct and complete
- Media file format is supported
- Server has CORS enabled

### Q: Can I share the same media multiple times?
**A:** Yes! Click Share as many times as you want. Each share creates a new message.

### Q: Does the media need to be downloaded?
**A:** No! Media streams directly from the URL. No download needed.

### Q: Can I edit the playlist?
**A:** Yes! Click the trash icon (🗑️) to remove items from the playlist.

### Q: Is the playlist saved?
**A:** Yes! Playlists are saved locally for each contact.

### Q: What if I close the modal?
**A:** Your playlist is saved! Click the 👁️ icon again to see it.

### Q: Can I share documents?
**A:** Yes! Select "Document" type and paste a PDF/DOC link.

### Q: Does it work on mobile?
**A:** Yes! Fully responsive on all devices.

## 🚀 Pro Tips

1. **YouTube Shortcuts**
   - Use short links: `youtu.be/VIDEO_ID`
   - Or full links: `youtube.com/watch?v=VIDEO_ID`
   - Both work automatically!

2. **Organize Playlists**
   - Add titles that describe the content
   - Remove old items to keep it clean
   - Create themed playlists for different topics

3. **Share Efficiently**
   - Add multiple items first
   - Then share one by one
   - Or share the same item multiple times

4. **Better Audio Quality**
   - Use direct MP3 links
   - Avoid compressed formats if possible
   - Test URL before sharing

5. **Video Tips**
   - YouTube videos are most reliable
   - Direct MP4 files work great too
   - Check video is publicly accessible
