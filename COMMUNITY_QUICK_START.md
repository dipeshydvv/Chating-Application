# Community Feature - Quick Start ⚡

## Access Communities
Click the **👥 Communities** icon in the left sidebar

---

## Quick Steps

### 1️⃣ Create Community
```
Click "New Community" 
→ Enter name & description
→ Click "Create Community"
```

### 2️⃣ Add Members
```
Select community
→ Click "Add Member"
→ Enter username
→ Click "Add"
```

### 3️⃣ Chat in Community
```
Select community
→ Click "Open Chat"
→ Type message
→ Press Enter or click Send
```

### 4️⃣ Manage Members
```
Select community
→ View members list
→ Click X to remove member
→ Click "Delete" to remove community
```

---

## Features at a Glance

| Feature | Description |
|---------|-------------|
| 👥 Create | Create communities with name & description |
| ➕ Add Members | Add members by username/email |
| ➖ Remove Members | Remove members from community |
| 💬 Chat | Send messages to all community members |
| 🔍 Search | Search communities by name |
| 📊 Stats | View member count and creation date |
| 🗑️ Delete | Delete community (creator only) |
| 💾 Save | All data persists automatically |

---

## Permissions

**Creator Can:**
- Add members
- Remove members
- Delete community
- Send messages

**Members Can:**
- Send messages
- View members
- View details

---

## Example: Create Study Group

1. Click Communities icon
2. Click "New Community"
3. Name: "Physics Study Group"
4. Description: "For physics students"
5. Click "Create Community"
6. Click "Add Member"
7. Add: "student1", "student2", "student3"
8. Click "Open Chat"
9. Start studying! 📚

- ✅ Communities saved in localStorage
- ✅ Messages persisted automatically

- ✅ No data loss
