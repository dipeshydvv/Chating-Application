## 📱 Feature 1: Send Text Message

```
1. Open app → Click "Messages"
2. Select user from list
3. Type message in input field
4. Click "Send" button
5. ✅ Message appears in chat
6. Other user sees it instantly
```

## 🗑️ Feature 6: Delete Message

### Option A: Delete from Me

```
1. Hover over your message
2. Click three dots (⋮)
3. ✅ Menu opens
4. Click "Delete"
5. ✅ Submenu opens
6. Click "Delete from me"
7. ✅ Message deleted from your view
8. Other person still sees it
```

### Option B: Delete from Everyone

```
1. Hover over your message
2. Click three dots (⋮)
3. ✅ Menu opens
4. Click "Delete"
5. ✅ Submenu opens
6. Click "Delete from everyone"
7. ✅ Message deleted for all
8. Other person can't see it
```

**Time:** 30 seconds

## 💡 Pro Tips

### Tip 1: Use @ to Mention
- Type @ to see all users
- Type @a to filter by 'a'
- Press Enter to select
- Mention appears in message

### Tip 2: Persistent Menu
- Menu stays open when you click
- Click elsewhere to close
- No need to click three dots again

### Tip 3: Two Delete Options
- "Delete from me" = only you
- "Delete from everyone" = all users
- Choose based on privacy needs

### Tip 4: Meeting Codes
- Codes are case-insensitive
- Share code to invite others
- Anyone with code can join
- Copy button for easy sharing

### Tip 5: Reply to Messages
- Hover on message
- Click three dots
- Click Reply
- Original message quoted

## 🔧 Troubleshooting Quick Fixes

### Problem: Menu disappears
**Fix:** Menu now stays open - click elsewhere to close

### Problem: Can't delete message
**Fix:** Only sender can delete. Check if it's your message

### Problem: Can't join meeting
**Fix:** Check meeting code is correct (case-insensitive)

### Problem: Photo not uploading
**Fix:** Check file size (max 10MB) and format (JPG, PNG, etc.)

### Problem: Video not uploading
**Fix:** Check file size (max 50MB) and format (MP4, MOV, etc.)

### Problem: Messages not syncing
**Fix:** Check internet connection and refresh page
