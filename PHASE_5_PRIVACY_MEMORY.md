## 🔥 1. Self-Destructing Messages

**What it does:** Send messages that automatically delete after a set time with screenshot detection alerts.

**How to access:**
- Click the 🔥 (flame) icon in the top header
- Type your secret message
- Set auto-delete timer (5s, 10s, 30s, 1m, 5m)
- Click "Send Self-Destructing Message"
- Get alerted if someone takes a screenshot

**Features:**
- **Message Control:**
  - Auto-delete after timer expires
  - View status tracking (viewed/not viewed)
  - Screenshot detection
  - Real-time countdown timer
  - Progress bar visualization

- **Screenshot Alerts:**
  - Instant notification if screenshotted
  - Timestamp of screenshot
  - Alert history
  - Auto-dismiss after 5 seconds

- **Timer Options:**
  - 5 seconds (ultra-fast)
  - 10 seconds (quick)
  - 30 seconds (standard)
  - 1 minute (extended)
  - 5 minutes (long-term)

**Key Benefits:**
- ✅ Maximum privacy
- ✅ Screenshot protection
- ✅ Sensitive info safety
- ✅ Confidential communication
- ✅ Trust building

**Use Cases:**
- Share passwords securely
- Send confidential information
- Share personal secrets
- Temporary sensitive data
- Privacy-first communication

**Example:**
```
Self-Destructing Message:
Message: "My password is..."
Timer: 10 seconds
Status: Viewed ✓
Screenshot Detected: ⚠️ Alert sent!
Time Left: 5 seconds ⏳
```

## 📞 2. Fake Incoming Call Mode

**What it does:** Create a fake incoming call to quickly exit a conversation without being rude.

**How to access:**
- Click the 📞 (phone) icon in the top header
- Click "Start Fake Incoming Call"
- A random caller name appears
- Take a screenshot to show others
- Call auto-ends after 30 seconds

**Features:**
- **Caller Names:**
  - 17 pre-loaded realistic names
  - Mom, Dad, Boss, HR, Bank, Doctor, Police, etc.
  - Random selection each time
  - Professional and casual options

- **Call Interface:**
  - Real-time call duration
  - Caller name display
  - Speaker button
  - End call button
  - Professional UI

- **Call History:**
  - Complete call log
  - Timestamps
  - Duration tracking
  - Copy call details
  - Share with others

**Key Benefits:**
- ✅ Graceful exit from conversations
- ✅ No awkward explanations
- ✅ Screenshot-ready interface
- ✅ Realistic appearance
- ✅ Complete call history

**Use Cases:**
- Exit boring conversations
- Escape uncomfortable situations
- Take a break from chatting
- Create believable excuse
- Share with friends

**Example:**
```
Fake Call:
Incoming Call from: Boss
Duration: 00:15
[Screenshot taken]
[Share with friend]
"Sorry, I have to take this call!"
```
