## 🚀 How to Login

### Option 1: Login with Email

**Step 1:** Click "Login with Email"
```
[📧 Login with Email]
```

**Step 2:** Enter your email
```
Email: your@email.com
```

**Step 3:** Click "Send OTP"
```
OTP will be sent to your email
(Shown in alert for testing)
```

**Step 4:** Enter the 6-digit OTP
```
OTP: 123456
```

**Step 5:** Create account or login
```
New User? → Create account
Existing User? → Enter password
```

---

## 🚀 Quick Tips

### Tip 1: OTP Testing
- OTP is shown in browser alert
- Copy and paste into OTP field
- Valid for 2 minutes

### Tip 2: Password Requirements
- Minimum 6 characters
- Can include numbers and symbols
- Must match confirmation

### Tip 3: Username Rules
- Minimum 3 characters
- Must be unique
- Can include letters, numbers, underscores

### Tip 4: Resend OTP
- Click "Resend OTP" after 2 minutes
- New OTP will be generated
- Previous OTP becomes invalid

## 🎯 Common Issues

### Issue: "Invalid email format"
**Solution:** Use valid email (user@example.com)

### Issue: "Invalid phone number"
**Solution:** Enter 10-digit phone number

### Issue: "Invalid OTP"
**Solution:** Check OTP in alert, enter exactly

### Issue: "Username already taken"
**Solution:** Choose different username

### Issue: "Passwords do not match"
**Solution:** Ensure both passwords are identical
